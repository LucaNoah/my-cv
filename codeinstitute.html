About Code Institute

Code Institute is a bootcamp program for full stack software development. I chose to study with Code Institute because there I could specialize in e-commerce. The positive feedback from former students have been very appealing to me & they offer a university credit-rated diploma.

What I Learned

In the Essentials Modules, I got introduced to front end software development and key UX principles. In the course of this I gained competency in the "programming" languages: HTML, CSS, JavaScript & Python.

I have also learned how to access databases and use tools like Bootstrap, Django, JQuery, PyTest and Jest. I have discovered how to test code and how agile development teams work.

In the end, I got a specialized eCommerce skill set by applying all the knowledge and techniques I have learned throughout the course. Therefore I have worked on an advanced end-to-end eCommerce website.

The course covers technical application, testing and deployment of web applications as well as holistic concepts like basics of good UI, principles of agile development, design thinking, web marketing and SEO.
